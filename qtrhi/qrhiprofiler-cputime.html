Detailed Description

Contains CPU-side frame timings.

Once sufficient number of frames have been rendered, the minimum, maximum, and average values (in milliseconds) from various measurements are made available in this struct queriable from QRhiProfiler::frameToFrameTimes() and QRhiProfiler::frameBuildTimes().

See also QRhiProfiler::setFrameTimingWriteInterval().
