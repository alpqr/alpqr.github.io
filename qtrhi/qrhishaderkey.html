Detailed Description

Specifies the shading language, the version with flags, and the variant.

A default constructed QRhiShaderKey has source set to SpirvShader and sourceVersion set to 100. sourceVariant defaults to StandardShader.

enum QRhiShaderKey::ShaderSource

Describes what kind of shader code an entry contains.

ConstantValueDescription
QRhiShaderKey::SpirvShader0SPIR-V
QRhiShaderKey::GlslShader1GLSL
QRhiShaderKey::HlslShader2HLSL
QRhiShaderKey::DxbcShader3Direct3D bytecode (HLSL compiled by fxc)

QRhiShaderKey::DxilShader5Direct3D bytecode (HLSL compiled by dxc)
QRhiShaderKey::MetalLibShader6Pre-compiled Metal bytecode

enum QRhiShaderKey::ShaderVariant

Describes what kind of shader code an entry contains.

ConstantValueDescription
QRhiShaderKey::StandardShader0The normal, unmodified version of the shader code.
QRhiShaderKey::BatchableVertexShader1Vertex shader rewritten to be suitable for Qt Quick scenegraph batching.
