Detailed Description

Describes the blend state for one color attachment.

Defaults to color write enabled, blending disabled. The blend values are set up for pre-multiplied alpha (One, OneMinusSrcAlpha, One, OneMinusSrcAlpha) by default.

Not having any TargetBlend specified in QRhiGraphicsPipeline::setTargetBlends() disables blending and is a convenient shortcut when having only one color attachment and no blending is desired. Otherwise a TargetBlend for each color attachment is expected.
