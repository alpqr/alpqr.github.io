Detailed Description

Command buffer resource.

Not creatable by applications at the moment. The only ways to obtain a valid QRhiCommandBuffer are to get it from the targeted swapchain via QRhiSwapChain::currentFrameCommandBuffer(), or, in case of rendering compeletely offscreen, initializing one via beginOffscreenFrame().
